# Архив анализов и отчетов

Эта папка содержит исторические анализы, отчеты и проверки.

## Структура:
- **logs/** - Анализы логов торговых сессий
- **problems/** - Анализы проблем и их решений
- **fixes/** - Отчеты об исправлениях
- **reports/** - Проверки и тесты
- **parameters/** - Анализы параметров

**Дата архивации:** 21.11.2025

Все эти файлы были перемещены из корня проекта для улучшения организации.
