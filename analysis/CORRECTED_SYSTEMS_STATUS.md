## 🔴 РЕАЛЬНЫЕ ПРОБЛЕМЫ

### 1. Breakeven (безубыток) — ОТСУТСТВУЕТ
**Статус:** Не реализовано

**Поиск:**
```python
breakeven          # Не найдено
break_even         # Не найдено
move_sl_to_entry   # Не найдено
```

**Проблемный сценарий:**
```
Вход LONG @ $95,000
SL установлен @ $94,000 (-1.0%)
Цена растёт до $96,425 (+1.5%)
TSL активируется, подтягивается до $95,900 (+0.95%)

НО: TSL никогда не достигнет $95,000 (точки входа)!
Потому что TSL = highest_price * (1 - trail%)

Цена разворачивается до $95,800
Закрытие по TSL @ $95,900 (+0.95%)
Комиссия: -0.04%
Итог: +0.91% при том, что был момент +1.5%

При этом если бы цена упала до $94,500 перед ростом — 
закрытие было бы в убыток, несмотря на то что позиция была в плюсе!
```

**Риск:**
- Позиция была в прибыли +1.5%
- Разворот → закрытие +0.91%
- Потеря 0.59% накопленной прибыли
- При частых сделках — существенная сумма

**Приоритет:** Средний-Высокий

**Что нужно:**
```python
# В TrailingStopLoss или отдельным модулем
async def check_breakeven(self, position, current_price):
    profit_pct = calculate_profit(current_price)
    
    if profit_pct > breakeven_trigger:  # например, 0.8%
        if not self.breakeven_activated:
            # Подтянуть SL к точке входа + комиссия
            breakeven_price = entry_price * (1 + fee_buffer)
            
            # Обновить highest_price/lowest_price для TSL
            if side == "long":
                self.highest_price = max(self.highest_price, breakeven_price)
            else:
                self.lowest_price = min(self.lowest_price, breakeven_price)
                
            self.breakeven_activated = True
            logger.info(f"Breakeven activated: SL moved to {breakeven_price}")
```
